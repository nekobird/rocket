### rgbaString

Get or set rgba color string.

```typescript
const color = new Color('red');

// Returns rgba(1, 0, 0, 0)
console.log(color.rgbaString);

color.rgbaString = 'rgba(0, 0, 1, 0)';

// Returns 1
console.log(color.b);
```
